Primeiras tags e Textos

Primeiramente devemos saber utilizar as tags de h1 a h6 e a tag p para organizar nosso texto de maneira correta. Além disso cabe lembrar que essas tags devem se encontrar dentro da tag body, indicando que fazem parte do corpo da página.

Além da tag body usamos também a tag head antes para indicar informações como title e a tag meta para dizer qual o charset (conjunto de caracteres) que usaremos, que no caso do português é o UTF-8.


Listas Ordenadas e Não Ordenadas

Logo vimos que precisávamos montar saber montar listas, então aqui vai uma:

  1. Escolher se vai usar uma lista ordenada (ol) ou não ordenada (ul)
  2. Além da tag que inicia a lista usar a tag para cada item (li)
  3. Não esquecer de fechar cada uma das tags que abrir

Além disso temos alguns pontos a considerar:

  • Podemos alterar o tipo de ícones das listas ordenadas para: números romanos, letras maiúsculas ou letras minúsculas
  • Podemos alterar o tipo de ícones das listas não ordenadas para outras formas como: quadrado (square) ou triângulo (triangle)
  • Nas listas ordenadas podemos usar a propriedade start para começar a lista de algum ponto específico
